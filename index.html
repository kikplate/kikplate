Deploy Kikplate on Kubernetes

Production-ready Helm chart for self-hosted boilerplate templates. Install with a single command.

Get Started View on GitHub

Installation

Login to GHCR

helm registry login ghcr.io -u moeidheidari -p YOUR_GITHUB_TOKEN

Add Repository

helm repo add kikplate oci://ghcr.io/moeidheidari/helm-charts

Install Chart

helm install kikplate kikplate/kikplate \ --namespace kikplate \ --create-namespace

Configure Credentials

helm install kikplate kikplate/kikplate \ --set secrets.jwtSecret=$(openssl rand -base64 32) \ --set secrets.sso.githubClientSecret=YOUR_GITHUB_SECRET \ --namespace kikplate
Need more options? View the complete values.yaml for all configuration options.
