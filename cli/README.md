# Kikplate CLI

A small standalone Go CLI built with Cobra.

The command name is:

```bash
kikplate <command>
```

For example:

```bash
kikplate hello
kikplate help
```

## Run

```bash
go run . --help
```

## Build

```bash
go build -o kikplate .
./kikplate --help
```

## Commands

```bash
go run . hello
go run . help
go run . help hello
```
